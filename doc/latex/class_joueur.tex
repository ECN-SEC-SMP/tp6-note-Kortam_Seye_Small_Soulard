\doxysubsection{Detailed Description}
Représente un joueur dans le jeu Monopoly. 

Chaque joueur possède des attributs tels que le nom, le solde, la position sur le plateau, les cartes de liberté, et la capacité à acheter des propriétés comme les terrains, les gares, etc.
