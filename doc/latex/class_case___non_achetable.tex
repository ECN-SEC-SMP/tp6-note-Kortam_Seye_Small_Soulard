\doxysubsection{Detailed Description}
Une classe représentant une case non achetable dans le jeu. 

Les cases non achetables ont un type spécifique (par exemple, Départ, Prison, etc.) et déclenchent des actions spécifiques lorsqu\textquotesingle{}un joueur atterrit dessus.
