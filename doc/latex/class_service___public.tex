\doxysubsection{Detailed Description}
Représente une case de service public dans le jeu de Monopoly. 

Cette classe hérite de {\ttfamily \doxylink{class_case___achetable}{Case\+\_\+\+Achetable}} et gère les interactions spécifiques aux services publics, tels que les loyers variables basés sur les dés et le nombre de services possédés par le propriétaire.
